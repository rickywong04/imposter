New Definition

Imposter MVP

A minimalist hybrid of asymmetric information and social bluffing. The digital system enforces rules; the game is the human conversation.

Core Loop Mechanics

The fundamental game cycle designed for 3-8 players. The system acts as the moderator, distributing differential information to create social friction.

Lobby & Roles

System picks a hidden Topic & Word. One random player is assigned Imposter.

Information Gap

Crew sees: Breakfast / Pancakes. Imposter sees: Breakfast / ???.

Turn-Based Flow

Active State

1. Random start player selected.
2. Each player submits ONE word to prove knowledge.
3. Imposter must blend in using only the category context.

Bare Minimum Specs

Technical Foundation

  • Lobby system with 4-char join codes
  • Hardcoded Word/Topic database (JSON)
  • State management for role assignment

Game Phase UI

  • Card Reveal: "WORD: [X]" or "???"
  • Voting Screen (Click player to accuse)
  • Simple Results (Imposter Caught / Wins)

Out of Scope: No chat system (voice external), no animations, no user profiles, no scoring persistence.
